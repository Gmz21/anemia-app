**AnemiaCare** es una aplicación web interactiva desarrollada en Python que permite analizar la distribución geográfica de pacientes con anemia en el departamento de San Martín y optimizar rutas de atención médica mediante algoritmos de grafos.
Este proyecto fue desarrollado en el contexto del curso de Complejidad Algorítmica, aplicando conceptos de optimización y estructuras de datos en un caso práctico con impacto en el ámbito de la salud.
